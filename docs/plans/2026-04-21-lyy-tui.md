# LYY TUI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the per-thread claude-pane model with a standalone React Ink TUI running in the right zellij pane, bridging chat ↔ main Claude ↔ peer conversation via explicit @Claude injection and `suggest_reply` callback.

**Architecture:** New `packages/tui/` React Ink app connects to the existing daemon's mcp.sock. Daemon gets a `subscribe` long-lived IPC channel that pushes events (message:new, suggest_reply). TUI does its own zellij shell calls to paste prompts into the left claude pane. Main Claude gets a new `suggest_reply` MCP tool that posts drafts back to the TUI. Old `/pickup` + `spawn-thread` + `thread.ts` flow deleted.

**Tech Stack:** Node 20 ESM, TypeScript, React + `ink` 5.x, `ink-text-input`, vitest, zellij shell actions, existing monorepo (pnpm workspaces, biome).

**Design doc:** `docs/plans/2026-04-21-lyy-tui-design.md`

```tsx
import { Box, Text, useInput } from "ink";
import React, { useEffect, useState } from "react";
import type { Message } from "@lyy/shared";
import type { State } from "@lyy/daemon";
import { ThreadView } from "./thread-view.js";

type View = { kind: "list" } | { kind: "thread"; threadId: string };

export interface AppProps {
  initialState: State;
  fetchMessages: (threadId: string) => Promise<Message[]>;
  selfPeerId: string;
}

export function App({ initialState, fetchMessages, selfPeerId }: AppProps) {
  const [selected, setSelected] = useState(0);
  const [view, setView] = useState<View>({ kind: "list" });
  const [messages, setMessages] = useState<Message[]>([]);
  const threads = initialState.threads;

  useInput((_input, key) => {
    if (view.kind === "list") {
      if (key.upArrow) setSelected((i) => Math.max(0, i - 1));
      if (key.downArrow)
        setSelected((i) => Math.min(threads.length - 1, i + 1));
      if (key.return && threads[selected]) {
        setView({ kind: "thread", threadId: threads[selected].threadId });
      }
    } else {
      if (key.escape) setView({ kind: "list" });
    }
  });

  useEffect(() => {
    if (view.kind === "thread") {
      fetchMessages(view.threadId).then(setMessages);
    }
  }, [view, fetchMessages]);

  if (view.kind === "thread") {
    const t = threads.find((x) => x.threadId === view.threadId);
    if (!t) return <Text>thread not found</Text>;
    return (
      <ThreadView
        thread={{ threadId: t.threadId, shortId: t.shortId, peerName: t.peerName }}
        messages={messages}
        selfPeerId={selfPeerId}
      />
    );
  }

  return (
    <Box flexDirection="column">
      <Text bold>LYY · 📬 {initialState.unreadCount} unread</Text>
      {threads.map((t, i) => {
        const marker = i === selected ? "▶ " : "  ";
        return (
          <Text key={t.threadId}>
            {marker}#{t.shortId} @{t.peerName}  {t.lastBody}
          </Text>
        );
      })}
    </Box>
  );
}
```

**Step 1: Decide the wire protocol additions**

Response frame unchanged: `{ id: number; result: unknown }` or `{ id: number; error: string }`.
**New event frame**: `{ type: "event"; event: string; payload: unknown }`. Sent unilaterally from daemon to a subscribed socket. Client distinguishes by presence of `type`.

**Step 2: Add `EventBus` inside `McpIpcServer`**

Track the sockets that called `subscribe`. When router wants to push, it calls `deps.eventBus.emit(event, payload)`, EventBus writes to each registered socket.

Add class field + methods at top of McpIpcServer:

```ts
private readonly subscribers = new Set<Socket>();

private pushToSubscribers(event: string, payload: unknown): void {
  const frame = `${JSON.stringify({ type: "event", event, payload })}\n`;
  for (const s of this.subscribers) {
    try {
      s.write(frame);
    } catch {
      this.subscribers.delete(s);
    }
  }
}
```

**Step 1: Add subscribe helper in `ipc.ts`**

The existing `McpIpcClient.call` is single-shot. For subscribe we need a raw socket that stays open and emits events. Add a second client:

```ts
import { createConnection } from "node:net";
import { DEFAULT_MCP_SOCK } from "@lyy/daemon";

export type EventHandler = (event: string, payload: unknown) => void;

export function subscribe(
  onEvent: EventHandler,
  sockPath: string = DEFAULT_MCP_SOCK,
): () => void {
  const socket = createConnection(sockPath, () => {
    socket.write(`${JSON.stringify({ id: 1, method: "subscribe" })}\n`);
  });
  let buffer = "";
  socket.on("data", (chunk) => {
    buffer += chunk.toString("utf8");
    let nl = buffer.indexOf("\n");
    while (nl !== -1) {
      const line = buffer.slice(0, nl);
      buffer = buffer.slice(nl + 1);
      if (line.trim()) {
        try {
          const frame = JSON.parse(line) as
            | { id: number; result: unknown }
            | { id: number; error: string }
            | { type: "event"; event: string; payload: unknown };
          if ("type" in frame && frame.type === "event") {
            onEvent(frame.event, frame.payload);
          }
        } catch {}
      }
      nl = buffer.indexOf("\n");
    }
  });
  return () => socket.destroy();
}
```

```ts
/** Toggle every `intervalMs` ms. Stable across renders. */
export function useBlink(intervalMs = 500): boolean {
  const [on, setOn] = useState(true);
```

```ts
import { spawn } from "node:child_process";

export interface BuildPromptInput {
  threadShortId: number;
  peerName: string;
  history: { sentAt: string; fromPeer: string; body: string }[];
  selfPeerId: string;
  question: string;
}

export function buildClaudePrompt(input: BuildPromptInput): string {
  const lines = input.history.map((m) => {
    const who = m.fromPeer === input.selfPeerId ? "me" : input.peerName;
    const time = m.sentAt.replace("T", " ").slice(0, 16);
    return `[${time}] ${who}: ${m.body}`;
  });
  return `You are in LYY thread #${input.threadShortId} with @${input.peerName}. Help me craft a reply.

History:
${lines.join("\n")}

My question: ${input.question}`;
}

/**
 * Write `text` into the left zellij pane (the user's main claude).
 * Assumes TUI is running inside zellij in the right pane.
 *
 * Implementation: focus left, write-chars, leave focus on left so the
 * user sees their prompt landed.
 */
export async function injectIntoClaudePane(text: string): Promise<void> {
  if (!process.env.ZELLIJ) {
    throw new Error("inject requires running inside zellij");
  }
  await run("zellij", ["action", "move-focus", "left"]);
  // write-chars may interpret \n as submit; test this first and switch to
  // bracketed paste if needed (see design doc §4.1).
  await run("zellij", ["action", "write-chars", text]);
}

function run(cmd: string, args: string[]): Promise<void> {
  return new Promise((resolve, reject) => {
    const proc = spawn(cmd, args, { stdio: "ignore" });
    proc.on("error", reject);
    proc.on("exit", (code) =>
      code === 0 ? resolve() : reject(new Error(`${cmd} exited ${code}`)),
    );
  });
}
```
