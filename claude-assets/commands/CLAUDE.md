# claude-assets/commands

Markdown slash-command templates for Claude Code. Installed by `lyy init` into `~/.claude/commands/`.

## Files

| File           | What                                                                                        | When to read                                            |
| -------------- | ------------------------------------------------------------------------------------------- | ------------------------------------------------------- |
| `inbox.md`     | `/inbox` — tell Claude to summarize unread peer messages via the `list_inbox` MCP tool      | Changing how unread summaries are framed                |
| `reply.md`     | `/reply` — help the user draft a reply to a specific thread                                  | Reply ergonomics / tone guidance                        |
| `send-to.md`   | `/send-to` — walk the user through sending a message to a peer                               | Send flow ergonomics                                    |
